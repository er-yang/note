# https

**对称加密**：指的就是加、解密使用的同是一串密钥，所以被称做对称加密

**非对称加密**：指的是加、解密使用不同的密钥，一把作为公开的公钥，另一把作为私钥。公钥加密的信息，只有私钥才能解密。反之，私钥加密的信息，只有公钥才能解密。 通讯双方交换自己的公钥，保留自己的私钥，双方用对方的公钥来加密给对方发送的信息，接收方在收到信息后，用自己的私钥来解密。
